    # 安装部署

    首先需要准备：

    • Node.js
    • Replicate Api Token

    Replicate Api Token 的获取也很简单，进入如下地址可以直接获取：

    https://replicate.com/
    1

    拷贝一下代码：

    git clone https://github.com/replicate/scribble-diffusion.git
    1

    在项目目录下新建 .env.local 配置文件：

    echo "REPLICATE_API_TOKEN=<your-token-here>" > .env.local
    1

    安装依赖：

    npm install
    1

    运行启动：

    npm run dev
    1

    浏览器访问：

    http://localhost:3000
    1
